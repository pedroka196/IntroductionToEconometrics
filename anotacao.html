Autocorrelao

A j-sima autocovarincia de uma srie \(Y_t\) a covarincia entre Y_t e sua j=rsima defasagem (lag), ou seja:

\[cov(Y_t,Y_{t-j})\] \[\rho_j = corr(Y_t,Y_{t-j}) = \frac{cov(Y_t,Y_{t-j})}{\sqrt{var(Y_t) var(Y_{t-j}) }}\]

14.3 Modelos autorregressivos

Modelo autorregressivo de primeira ordem

Se o objetivo prever o futuro, nada melhor para isso que o perodo imediatamente anterior. Sendo assim, se o objetivo for ver a variao da inflao deste trimestre para o prximo, pode ver se a inflao aumentou ou diminuiu no trimestre passado.

Uma forma de testar isso, seria rodando um MQO de \(\Delta INF_t\) sobre \(\Delta INF_{t-1}\).

\[\Delta INF_t = \beta_0 + \beta_1 \Delta INF_{t-1} + u_t\] Essa equao acima chamada de autorregresso de primeira ordem. Uma regresso de uma srie sobre sua prpria defasagem. Para isso, usamos a funo ar.ols.

INF.TAXA.ols.1 <- ar.ols(INF.Taxa["1962::2006"],
       order.max = 1,
       demean = F,
       intercept = T)

Erros de previso

Em geral, \(\hat{Y}_{t|t-1}\) representar a previso de \(Y_t\) baseada em informaes at o perodo \(t-1\). Assim, o modelo \(AR(1)\) :

\[ \hat{Y}_{t|t-1} = \hat{\beta_0} + \hat{\beta_1} Y_{t-1}\]

O erro de previso o erro cometido pela previso. Ou seja, a diferena entre o valor \(Y_t\) que ocorreu e o valor previsto em \(Y_{t-1}\).

\[ erro \space de \space previso = Y_{t}-\hat{Y}_{t|t-1} \]

Previso vs valores previstos

A previso no um valor previsto de MQO, e o erro de previso no um resduo de MQO. * Valores previstos de MQO so calculados para as observaes de dentro da amostra utilizada. * Previso feita para dados de fora" da amostra, feita para alguma data alm da data base de dadis utilizada.

Raiz do erro de previso quadrtico mdio ( root mean squared forecast error - RMSFE)

uma medida do tamanho do erro de previso, isto , da magnitude de um erro tpico cometido utilizando um modelo de previso.

\[RMSFE = \sqrt{E\left[\left(Y_{T+1} - \widehat{Y}_{T+1\vert T}\right)^2\right]}\]
